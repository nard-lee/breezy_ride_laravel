This is laravel with livewire, tailwind.

To begin, first you need:
 -- xampp
 -- composer
 -- node

note: dont forget to add php, mysql to your environment variables.

to start open project, then pop two terminals and run this command,

before next step type in terminal: php artisan migrate 
note: create database first look for it on .env file
find this:
  DB_CONNECTION=mysql
  DB_HOST=127.0.0.1
  DB_PORT=3306
  DB_DATABASE=breezy_ride
  DB_USERNAME=root
  DB_PASSWORD=

on first terminal type: npm run dev,
  -- this is for node js to compile tailwind, js files etc,
  -- if you want bootstrap css then we migh not need this well use a cdn (dont be confused by bootstrap folder its not same)

second terminal type: php artisan serve
  -- this will run the server for main project
     after running it will show the link usually:
        -- localhost:port ex. localhost:8000
        -- 127.0.0.1:port ex. 127.0.0.1:8000
     open in browser, youll see login and signup link 
     click on one and itll show the page

youll see google sign in as well.
